# Automated Installation Script - Jchock93

Script de automatización en Batch para el despliegue rápido de software esencial en Windows utilizando **Winget**.

## 🚀 Características
- Instalación desatendida de: Chrome, Brave, WinRAR, VLC, Google Earth, RustDesk y WireGuard.
- Ejecución automática con privilegios de Administrador.
- Menú interactivo con arte ASCII personalizado.
- **Sistema de Skip:** Contador de 5 segundos para cancelar instalaciones individuales.

## 🛠️ Requisitos
- Windows 10 o 11 (con Winget instalado).
- Conexión a Internet.

## 🖥️ Uso
1. Descarga el archivo `install.bat`.
2. Ejecútalo como administrador.
3. Presiona ENTER y decide si quieres saltar alguna app durante el conteo.
